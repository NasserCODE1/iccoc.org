# ICCOC – Angular v20 i18n Starter (Zip)
هذه الحزمة تحتوي **src/** الجاهز مع ملفات الترجمة والمكوّنات. لا تشمل ملفات Angular الافتراضية (angular.json, tsconfig…).

## طريقة الاستخدام
1) إنشاء مشروع Angular v20 فارغ:
```bash
npm i -g @angular/cli@20
ng new iccoc --standalone --routing --style=scss
cd iccoc
npm i @ngx-translate/core @ngx-translate/http-loader
```

2) انسخ محتويات هذا المجلد `src/` فوق `src/` في مشروعك (استبدال الملفات).
3) شغّل:
```bash
ng serve --open
```
4) بناء وإخراج الإنتاج:
```bash
ng build --configuration production
# ستجد المخرجات في: dist/iccoc/browser
```

> ملاحظة: المبدّل يغيّر الاتجاه RTL/LTR تلقائيًا. يمكنك توسيع الصفحات ونقل باقي المحتوى بسهولة.
